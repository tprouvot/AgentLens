# Changelog

## 0.3.0

- Agent Graph visualization showing sub-agent communication paths
- Finite State Machine diagram for each sub-agent
- Step-by-step inspector with keyboard navigation (arrow keys)
- Filter toolbar: variable updates, reasoning, node entry, enabled tools, internal vars
- CodeLens integration for `.sfdx/agents/**/traces/*.json` files
- Custom editor provider for JSON trace files
- Context menu "Open with AgentLens" for JSON files
- Native VS Code theme support (light, dark, high-contrast)
- Paste, upload, and drag-and-drop trace loading
- Copy analysis report to clipboard as Markdown
